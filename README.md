# Proyecto: Sidequest Journal (iOS)

Carpeta de proyecto para el MVP de una app iOS (iOS 17) de hábitos/journal con retos diarios.

## Lectura rápida (para retomar contexto)
- **Índice del proyecto:** `PROJECT_MEMORY.md` (léelo primero)
- **README ampliado:** `README_EXPANDED.md`

## Qué hay aquí
- `docs/` documentación del MVP (alcance, arquitectura, backlog, modelo de datos)
- `data/challenges.json` catálogo inicial de categorías/medallas/retos
- `app/` proyecto Xcode y código iOS

## Decisiones (2026-02-21)
- Offline-first
- iOS 17
- Reto diario asignado por **hora fija configurable** (default sugerido 07:00)
- Anti-repetición: 30 días
- Evidencia obligatoria (texto/foto/video/audio) con visibilidad privada/pública
- Medallas por reto y categorías
